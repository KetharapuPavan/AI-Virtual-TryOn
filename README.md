# Autonomous AI-Powered Virtual Try-On SaaS Platform

An AI-powered virtual try-on system that allows users to upload images and visualize fashion apparel digitally using Computer Vision, Pose Estimation, and Generative AI.

## Features

- Body Detection
- Pose Estimation
- Fashion Visualization
- Real-Time Processing
- Cloud Deployable
- FastAPI Backend

## Technologies

- Python
- FastAPI
- OpenCV
- MediaPipe
- Pillow
- NumPy

## Installation

```bash
pip install -r requirements.txt
```

Run:

```bash
python app.py
```

Open:

http://127.0.0.1:8000/docs
